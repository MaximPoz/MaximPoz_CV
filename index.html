em - это курсив ( i больше не используют )

strong - жирный шриф ( b больше не использовать )

target="_blank" - используется для того что бы

ol type="I" - римские цифры в списке

ul type='square' - квадраты в списке

a href="Youtube.html" - ссылка на наш файл в котором лежит a href="https://www.youtube.com/watch?v=DOEtVdkKwcU"> HTML для Начинающих - Практический Курс [2021] таким образом мы переходим на новую страницу которую создали и видим там ul список в котором есть сссылка на урок.

table>tr>td*3 - создают таблицу 3 на 3 см.31-42 строки атрибуты "html table mdn". tr - строка, td - столбец.

cellspacing='15' - отступы

input type="submit" value="Связаться" - кнопка с дефолтным значением "Отправить", его можно изменить value="Связаться"

value="" может изменять значение так же и в label (после placeholder)

form action="/" - элемент позволяющий создать эл.управления и отправить инф. на сервер
